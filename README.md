# Job Application Tracker (Python)

A simple command-line job application tracker built with Python and SQLite.

## Features
- Add job applications
- Store data persistently using SQLite
- Clean multi-file project structure

## File Structure
- `app.py` – Application entry point
- `db.py` – Database connection and setup
- `models.py` – Job data model
- `utils.py` – Helper utilities
- `README.md` – Project documentation

## How to Run
```bash
python app.py
```

## Example Output
```
Job Applications:
----------------------------------------
Google          | Software Engineer     | Applied
Amazon          | Backend Engineer      | Interview
```
